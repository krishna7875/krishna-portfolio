# Krishna Bansode - Developer Portfolio

A professional, high-performance portfolio website built for a Senior Backend/Laravel Developer.
Designed to be lightweight, SEO-friendly, and mobile-first using modern web technologies.

## Tech Stack

- **Framework:** React 18 (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Typography:** Inter & JetBrains Mono

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Features

- **Performance First:** Minimal dependencies, optimized assets, and clean DOM structure.
- **SEO Optimized:** Meta tags, Open Graph support, and semantic HTML5.
- **Mobile Responsive:** Fully adaptive layout for all device sizes.
- **Dark Mode Accents:** Professional light theme with deep slate/navy contrasts.

## Project Structure

- `src/components/` - Reusable UI sections (Hero, About, Skills, etc.)
- `src/components/ui/` - Shared UI wrappers (Section.jsx)
- `src/index.css` - Tailwind imports and global styles

## Customization

- Update `src/components/Contact.jsx` with your actual email and phone links.
- Replace `src/components/Hero.jsx` resume link with your actual PDF path.
